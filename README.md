# Anuraj Makeover Website

This is a PHP + MySQL based salon booking website.

## Features
- Booking form
- Admin dashboard
- Database connection
- Email notification (SMTP ready)

## Folder Structure
- /admin → Admin panel
- booking.php → Booking handler
- db-sample.php → Database config example

## Note
Database and credentials are not included for security reasons.
